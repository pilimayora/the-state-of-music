The state of music is a coursework presented by Group 7 - International Super Friends (Cleo Au, William Low, Pili Mayora, Richa Sud and Matthew Sutton) for UCL CASA's module CASA0009 Spatial Data Capture, Storage and Analysis.

The website utilises music performance information (name, event location and genre) extracted from ticketmaster's and eventbrite's API for the next 30 days.

The genre of playing artists were then match with the one classified by Spotifyâ€™s API to ensure higher accuracy of the events' genre.

All events are then grouped by genre according to a dictionary created from web-scraping the List of Music Styles in Wikipedia.
