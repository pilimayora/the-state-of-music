The State of Music

The state of music is a coursework presented by Group 7 - International Super Friends for UCL CASA's module CASA0009 Spatial Data Capture, Storage and Analysis. Feel free to check our project on Github.

The website utilises music performance information (name, event location and genre) extracted from ticketmaster's and eventbrite's API for the next 30 days.

The genre of playing artists were then match with the one classified by Spotifyâ€™s API to ensure higher accuracy of the events' genre.

All events are then grouped by genre according to a dictionary created from web-scraping the List of Music Styles in Wikipedia.

The State of Music's Search API

Resource URL

http://35.185.97.77/search

Parameters

  • genre: Filter by genre (ex: pop)
  • admin_level: Filter by administrative level. Admin level 1: states; admin level 2: counties and parishes; admin level 3: venues.
  • state: Filter by state (ex: CA)
  • county: Filter by county (ex: San Francisco)

Examples

Get music stats from all venues in San Francisco County, California: /search?admin_level=3&state=CA&county=San%20Francisco

Get music stats for all counties playing pop in Nevada: /search?genre=pop&admin_level=2&state=NV

Get aggregated music stats for the state of New York: /search?admin_level=1&state=NY

Use this API responsibly. We don't currently have rate limiting implemented; be nice.
